# 🏦 Sistema Bancário em Java (Maven)

Sistema bancário desenvolvido em **Java**, utilizando **Maven**, com foco no estudo de **orientação a objetos**, **persistência de dados** e boas práticas de organização de projetos.

O projeto simula operações básicas de um banco, permitindo a manipulação de contas e transações financeiras em um ambiente controlado.

---

## 📌 Objetivo do Projeto

Este projeto foi criado com fins **educacionais**, visando:
- praticar Java orientado a objetos
- compreender a estrutura de um projeto Maven
- trabalhar com persistência de dados
- aplicar validações e regras de negócio

---

## 🛠 Tecnologias Utilizadas

- Java  
- Maven  
- JDBC  
- SQLite  
- Git e GitHub  

---

## ✨ Funcionalidades

- Criar contas bancárias  
- Realizar depósitos  
- Realizar saques  
- Consultar saldo  
- Persistência de dados em banco local
